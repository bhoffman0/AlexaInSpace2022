    <p class="overview">This unit focuses on <b><i>Computing Systems and Networks</i></b>, one of the big ideas in computer science. The App Inventor lesson in this unit shows how to use the Internet in apps, including the ability to send text messages over <span class="hover vocab yui-wk-div" data-id="wifi">wifi</span>. The CS Principles lessons focus on the Internet, how it works, how it enables innovation and collaboration, and security concerns for using it. In this unit, you will complete a series of activities using <span class="hover vocab yui-wk-div" data-id='network'>network</span> administration software tools such as <span class="hover vocab yui-wk-div" data-id='ping'>Ping</span> and <span class="hover vocab yui-wk-div" data-id="traceroute">traceroute </span> as well as a <span class="hover vocab yui-wk-div" data-id='DNS'>Domain Name System (DNS)</span> simulator app to explore how we communicate on the Internet with <span class="hover vocab yui-wk-div" data-id="IP Address">IP Addresses</span>.
    </p>
